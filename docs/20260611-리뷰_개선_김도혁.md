## 1. [필수·최우선] 빌드 복구 — getLikeList 호출부 수정 ✅

`getLikeList`가 `{ likedIds, currentUserId }` 객체를 반환하도록 바뀐 것에 맞춰, 받는 쪽을 모두 구조분해로 수정.

**수정 파일**
- `src/app/(user)/usedgoods/page.tsx`
- `src/app/(user)/job/page.tsx`
- `src/app/(user)/main/_components/popular/Popular.tsx`

```ts
// before
const [posts, likedIds] = await Promise.all([..., getLikeList("used_goods")]);

// after
const [posts, { likedIds }] = await Promise.all([..., getLikeList("used_goods")]);
```
