# Admin 글 관리 페이지 & 중고거래 예약중 기능

## 1. Admin 글 관리 페이지

### 개요
어드민 사이드바의 "글 관리" 메뉴를 활성화하고, 전체 게시글 목록을 조회·관리할 수 있는 페이지를 추가했습니다.

### 주요 기능
- 전체 게시글 목록 (제목, 카테고리, 작성자, 조회수, 작성일, 상태)
- 카테고리 탭 필터 (전체 / 중고거래 / 구인구직 / 방렌트 / 커뮤니티)
- 제목·작성자 키워드 검색
- 활성 ↔ 비공개 토글 (클릭 즉시 DB 반영, 낙관적 UI)
- 데스크탑: 테이블, 모바일: 카드 레이아웃
- 제목 클릭 시 상세 페이지 `_blank` 이동 (community 제외)

## 2. 중고거래 예약중 기능

### 개요
중고거래 게시글에 `is_reserved` boolean 상태를 추가하여, 채팅방에서 판매자가 예약중으로 변경할 수 있고 목록·상세 페이지에 예약중 표시가 나타나도록 구현했습니다.

### DB 변경
```sql
-- supabase/migrations/20260618100000_add_is_reserved_to_posts.sql
ALTER TABLE posts ADD COLUMN IF NOT EXISTS is_reserved boolean NOT NULL DEFAULT false;
```
> Supabase 대시보드 SQL Editor에서 직접 실행 필요

### 기능 흐름
1. 판매자가 채팅방 우상단 `...` 메뉴 → **예약중으로 변경** 클릭
2. `posts.is_reserved = true` DB 업데이트 (Server Action)
3. 중고거래 목록 카드 이미지 좌상단에 주황 "예약중" 배지 표시
4. 중고거래 상세 페이지 이미지에 반투명 오버레이 + "예약중" 텍스트 표시
5. 재클릭 시 **예약 취소** → `is_reserved = false`

### 버튼 노출 조건
- 채팅방 게시글이 `used_goods` 타입인 경우에만 표시
- 현재 로그인 유저가 판매자(sellerId)인 경우에만 표시
