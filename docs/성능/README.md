# 칸토 성능 관리

성능 **검사(측정·문제 발견)** 와 **개선(수정·전후 비교)** 문서를 모아두는 폴더입니다.
기준값·측정 방법론의 상세는 → [`../설계/20260630-성능감사-프레임워크.md`](../설계/20260630-성능감사-프레임워크.md)

## 폴더 구조

```
성능/
├── README.md      ← (이 문서) 전체 통합 대시보드
├── 성능검사/       ← 각자 측정한 검사 문서 (_템플릿.md 복사해서 작성)
└── 성능개선/       ← 개선 작업별 전/후 지표 문서 (_템플릿.md 복사해서 작성)
```

## 작업 흐름

1. **검사**: `성능검사/_템플릿.md`를 복사해 `이름-성능검사.md`로 만들고, 맡은 페이지를 측정해 채운다.
2. **통합**: 작성 후 아래 **통합 진행 현황** 표에 본인 페이지 결과를 한 줄씩 추가한다.
3. **개선**: 검사에서 나온 P0/P1 문제는 `성능개선/_템플릿.md`를 복사해 개선 작업 문서를 만들고, 전/후 지표를 기록한다.

> 측정 공통 규칙: **PSI · 배포 URL · 모바일+데스크탑 둘 다 · 2회 이상 대표값 · Lab 기준**
